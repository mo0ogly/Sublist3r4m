Sublist3r4m

Subdomain enumeration tool for penetration testers and security researchers. Powered by OSINT.

Python 3.9+ GPL-2.0 v1.0.0
View on GitHub Releases

Features

Multi-Engine Search

Enumerate subdomains via Google, Yahoo, Bing, Baidu, Ask, Netcraft, VirusTotal, ThreatCrowd, DNSdumpster, crt.sh, and more.

JARVIS Intelligence

Enhanced scanning engine with Certificate Transparency, SecurityTrails, retry logic, rate limiting, and multi-format export.

DNS Bruteforce

Integrated subbrute module with optimized wordlists and multi-process DNS resolution for fast subdomain discovery.

Owner Research

Advanced domain owner research with fuzzy matching (Levenshtein, Soundex, Jaro-Winkler), WHOIS lookups, and SQLite caching.

Port Scanning

Identify open TCP ports on discovered subdomains to map the attack surface.

GUI Interface

Advanced Tkinter GUI with real-time logging, result filtering, statistics, and multi-format export.

Quick Start

# Clone the repository
git clone https://github.com/mo0ogly/Sublist3r4m.git
cd Sublist3r4m

# Install dependencies
pip install -e ".[dev]"

# Enumerate subdomains
python sublist3r.py -d example.com

# With bruteforce and port scanning
python sublist3r.py -d example.com -b -p 80,443

# Run tests
make test

CLI Options

FlagDescription
-dTarget domain to enumerate
-bEnable bruteforce module
-pScan specific TCP ports (comma-separated)
-vVerbose mode (real-time results)
-tNumber of threads for bruteforce
-eSpecific engines (comma-separated)
-oOutput file path
